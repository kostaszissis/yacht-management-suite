## ΕΓΚΑΤΑΣΤΑΣΗ ΕΦΑΡΜΟΓΗΣ / APP INSTALLATION

Η εφαρμογή λειτουργεί ως PWA (Progressive Web App) και μπορεί να εγκατασταθεί σε οποιαδήποτε συσκευή.

The app works as a PWA (Progressive Web App) and can be installed on any device.

---

### 📱 ΚΙΝΗΤΟ (MOBILE PHONE)

#### iPhone (iOS)
**ΠΡΟΣΟΧΗ: Χρησιμοποιήστε ΜΟΝΟ Safari!**
**ATTENTION: Use ONLY Safari!**

1. Ανοίξτε το **Safari** (όχι Chrome!)
2. Πηγαίνετε στο: `https://yachtmanagementsuite.com`
3. Πατήστε το κουμπί **Κοινοποίηση** (τετράγωνο με βέλος ↑)
4. Επιλέξτε **"Προσθήκη στην οθόνη Αφετηρίας"**
5. Πατήστε **"Προσθήκη"**
6. Το εικονίδιο με την άγκυρα θα εμφανιστεί στην αρχική οθόνη

*Screenshot: Κουμπί κοινοποίησης Safari → Προσθήκη στην οθόνη Αφετηρίας*

#### Android Phone
**Χρησιμοποιήστε Chrome**
**Use Chrome**

1. Ανοίξτε το **Chrome**
2. Πηγαίνετε στο: `https://yachtmanagementsuite.com`
3. Θα εμφανιστεί banner **"Εγκατάσταση εφαρμογής"** - Πατήστε το!
4. Ή: Πατήστε τις 3 τελείες (⋮) → **"Εγκατάσταση εφαρμογής"**
5. Πατήστε **"Εγκατάσταση"**
6. Το εικονίδιο θα εμφανιστεί στην αρχική οθόνη

*Screenshot: Chrome menu → Εγκατάσταση εφαρμογής*

## ΟΔΗΓΙΕΣ ΓΙΑ ΠΛΟΙΟΚΤΗΤΕΣ - ΠΩΣ ΚΑΝΩ ΑΠΟΔΟΧΗ ΝΑΥΛΟΥ
## OWNER GUIDE - HOW TO ACCEPT A CHARTER

---

### ΒΗΜΑ 1: ΣΥΝΔΕΣΗ / LOGIN

1. Ανοίξτε την εφαρμογή (από το εικονίδιο με την άγκυρα)
2. Εισάγετε τον **κωδικό πλοιοκτήτη** σας
   - Μορφή κωδικού: `X2025`, `Y2025`, κτλ.
   - Ο κωδικός σας δόθηκε από τον διαχειριστή
3. Πατήστε **"ΕΙΣΟΔΟΣ"**
4. Θα δείτε **ΜΟΝΟ τα δικά σας σκάφη**

*Screenshot: Οθόνη login με πεδίο κωδικού*

---

### ΒΗΜΑ 2: ΕΙΔΟΠΟΙΗΣΗ ΝΕΟΥ ΝΑΥΛΟΥ / NEW CHARTER NOTIFICATION

Όταν υπάρχει νέο αίτημα ναύλου:

1. Θα δείτε **ειδοποίηση** στο dashboard σας
2. Η κατάσταση θα είναι: **"OPTION"** 🟡 (κίτρινο)
3. Θα βλέπετε:
   - Ημερομηνίες ναύλου (Check-in / Check-out)
   - Ποσό ναύλου
   - Όνομα πελάτη
   - Στοιχεία κράτησης

*Screenshot: Dashboard με νέο option σε κίτρινο status*

---

### ΒΗΜΑ 3: ΕΛΕΓΧΟΣ ΣΤΟΙΧΕΙΩΝ ΝΑΥΛΟΥ / REVIEW CHARTER DETAILS

Πριν αποδεχτείτε, ελέγξτε προσεκτικά:

| Στοιχείο | Ελέγξτε |
|----------|---------|
| 📅 **Ημερομηνίες** | Check-in και Check-out |
| 💶 **Ποσό Ναύλου** | Συμφωνημένο ποσό |
| 💵 **Προμήθεια** | Ποσοστό γραφείου |
| 👤 **Πελάτης** | Όνομα, εθνικότητα |
| 🛥️ **Σκάφος** | Σωστό σκάφος επιλεγμένο |

*Screenshot: Αναλυτικά στοιχεία κράτησης*

---

### ΒΗΜΑ 4: ΑΠΟΔΟΧΗ Ή ΑΠΟΡΡΙΨΗ / ACCEPT OR REJECT

Θα δείτε δύο κουμπιά:

| Κουμπί | Χρώμα | Ενέργεια |
|--------|-------|----------|
| **ΑΠΟΔΟΧΗ** | 🟢 Πράσινο | Αποδέχεστε τον ναύλο |
| **ΑΠΟΡΡΙΨΗ** | 🔴 Κόκκινο | Απορρίπτετε τον ναύλο |

**Για ΑΠΟΔΟΧΗ:**
1. Πατήστε το πράσινο κουμπί **"ΑΠΟΔΟΧΗ"**
2. Θα εμφανιστεί επιβεβαίωση
3. Πατήστε **"ΝΑΙ, ΑΠΟΔΕΧΟΜΑΙ"**

**Για ΑΠΟΡΡΙΨΗ:**
1. Πατήστε το κόκκινο κουμπί **"ΑΠΟΡΡΙΨΗ"**
2. Εισάγετε τον λόγο απόρριψης (προαιρετικά)
3. Πατήστε **"ΕΠΙΒΕΒΑΙΩΣΗ ΑΠΟΡΡΙΨΗΣ"**

*Screenshot: Κουμπιά αποδοχής/απόρριψης*

---

### ΒΗΜΑ 5: ΜΕΤΑ ΤΗΝ ΑΠΟΔΟΧΗ / AFTER ACCEPTANCE

Αφού αποδεχτείτε:

1. Η κατάσταση αλλάζει σε: **"OPTION ACCEPTED"** 🟡
2. Ο διαχειριστής λαμβάνει ειδοποίηση
3. Ο διαχειριστής επικοινωνεί με τον πελάτη
4. **Περιμένετε** την τελική επιβεβαίωση

⏳ *Σε αυτό το στάδιο η κράτηση είναι ακόμα προσωρινή*

*Screenshot: Status "Option Accepted" σε κίτρινο*

---

### ΒΗΜΑ 6: ΤΕΛΙΚΗ ΕΠΙΒΕΒΑΙΩΣΗ / FINAL CONFIRMATION

Η διαδικασία ολοκληρώνεται ως εξής:

| Βήμα | Ενέργεια | Κατάσταση |
|------|----------|-----------|
| 1 | Διαχειριστής πατάει **"ΚΛΕΙΣΙΜΟ OPTION"** | 🟡 RESERVATION |
| 2 | Εσείς επιβεβαιώνετε ξανά | 🟡 PENDING CONFIRM |
| 3 | Τελική επιβεβαίωση | 🟢 **CONFIRMED** |

**Όταν η κατάσταση γίνει CONFIRMED (πράσινο):**
- Η κράτηση είναι **οριστική**
- Δεν μπορεί να ακυρωθεί χωρίς επικοινωνία
- Οι ημερομηνίες είναι δεσμευμένες

*Screenshot: Final status "Confirmed" σε πράσινο*

## ΣΗΜΑΝΤΙΚΕΣ ΣΗΜΕΙΩΣΕΙΣ ΓΙΑ ΠΛΟΙΟΚΤΗΤΕΣ
## IMPORTANT NOTES FOR OWNERS

### ✅ ΤΙ ΜΠΟΡΕΙΤΕ ΝΑ ΚΑΝΕΤΕ:

- Να βλέπετε **ΜΟΝΟ τα δικά σας σκάφη**
- Να αποδέχεστε ή να απορρίπτετε options
- Να βλέπετε το ιστορικό κρατήσεων
- Να βλέπετε τα οικονομικά στοιχεία
- Να λαμβάνετε email ειδοποιήσεις

### ❌ ΤΙ ΔΕΝ ΜΠΟΡΕΙΤΕ ΝΑ ΚΑΝΕΤΕ:

- Να βλέπετε σκάφη άλλων πλοιοκτητών
- Να αλλάξετε στοιχεία κράτησης
- Να ακυρώσετε confirmed κράτηση μόνοι σας
- Να δημιουργήσετε νέες κρατήσεις

### 📧 EMAIL ΕΙΔΟΠΟΙΗΣΕΙΣ:

Θα λαμβάνετε email σε κάθε στάδιο:
1. ✉️ Νέο option - αίτημα αποδοχής
2. ✉️ Επιβεβαίωση αποδοχής σας
3. ✉️ Κλείσιμο option → Reservation
4. ✉️ Τελική επιβεβαίωση → Confirmed
5. ✉️ Υπενθύμιση πριν το check-in

### ⚠️ ΠΡΟΣΟΧΗ:

- **Μπορείτε να απορρίψετε ΜΟΝΟ στο στάδιο Option**
- Μόλις γίνει Confirmed, η κράτηση είναι δεσμευτική
- Για αλλαγές επικοινωνήστε με τον διαχειριστή
- Ελέγχετε τακτικά την εφαρμογή για νέα αιτήματα

```
🔴 CANCELLED        → Ακυρωμένο

```
